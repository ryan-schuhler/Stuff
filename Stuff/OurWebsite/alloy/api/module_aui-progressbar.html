Module: aui-progressbar

The ProgressBar Utility provides a visual means of showing progress of an ongoing operation. The ProgressBar can be enhanced via CSS styles to provide different colors, shapes and textures. The bar can move horizontally or vertically. The movement can be enhanced by using the Animation utility.
